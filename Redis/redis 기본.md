# Redis 

## 무엇이냐

    Remote Dictionary Server

    인 메모리 키 값 데이터 구조 스토어?

    싱글 스레드

    캐시의 용도로 흔히 사용된다


## 장점 

    빠르다!

    다양한 데이터 구조 집합을 제공

## 캐시(Cache)란?

    요청된 결과를 미리 저장하여, 추후 요청에 제공하는 것    

    빠르지만 용량은 작음

    L1, L2 캐시를 가지고 있음

    파레토의 법칙을 받음
        상위 20%가 전체의 80%를 해낸다
        캐시를 적용할 떄, 가장 많이 사용되는 20%를 캐싱한다면, 성능을 크게 올릴 수 있음
        Redis를 캐싱의 용도로 사용할 떄, 20%의 가장 많이 사용되는 데이터를 확인해서 캐싱하는 것이 중요
