# $()

## 사용법

    $(선택자).동작함수()

## $ 

    $ 기호는 jQuery를 의미. 

    즉, $() 과 jQuery()는 동일함. 

## 의미

    제이쿼리에 접근할 수 있게 해주는 식별자. 

    선택된 HTML 요소를 제이쿼리에서 이용할 수 있는 형태로 생성해 주는 역할. 

    선택자로는 HTML 태그 이름, CSS 선택자를 전달하여 특정 HTML 요소를 선택할 수 있음. 
    
    이렇게 $()함수로 생성된 요소를 제이쿼리 객체라고 함.
